# RCB IPL 2026 Analytics Dashboard

## Project Overview
Interactive Power BI dashboard analyzing RCB's IPL 2026 season performance.

## Business Problem
Cricket teams generate large volumes of match data. This project transforms raw scorecard data into actionable insights around team performance, batting, bowling, and tournament progression.

## Dashboard Features
- Match Summary Analysis
- Batting Performance
- Bowling Performance
- Points Table Tracking
- Net Run Rate Analysis
- Opponent-wise Insights

## Tools Used
- Power BI
- Excel
- DAX
- Data Modeling
